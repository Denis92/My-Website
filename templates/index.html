"Умный Дом. REST API"

Описание проекта: Данный проект предусматривает реализацию REST API для управления IoT устройствами такими как:

  1. Умный обогреватель
  2. Умная лампочка Wi-Fi Xiaomi
  3. Умная лампочка Zigbee Ikea
  4. Умный увлажнитель воздуха
  5. Автоматическая система автополива цветов
  6. Роботизированная система для патрулирования квартиры
Помимо управлния перечисленными устройствами сервис позволяет получать информацию о температуре и влажности в квартире.

Цель проекта:

  1. Изучить принципы построения Backend приложений и реализовать REST API для дальнейшей работы с Fronted приложением
  2. Разработать собственные IoT устройства, в рамках данного проекта, а именно
    1. Роботизированную систему для патрулирования квартиры
    2. Умный обогреватель
    3. Умный увлажнитель воздуха
    4. Автоматическая система автополива цветов

"Телеграмм Бот"

Цель проекта: Изучить принципы работы с API Телеграмма и разработать бота, который будет обладать следеющим функционалом:

  1. Показывать погоду в любом месте мира, как по текстовому запросу, так и по указанию на карте. Так же выдавать погоду по подписке на текущий день
  2. Отправлять напоминания
  3. Управлять рядом имеющихся IoT устройств
  4. Отобращать некоторую информацию о состоянии сервера
